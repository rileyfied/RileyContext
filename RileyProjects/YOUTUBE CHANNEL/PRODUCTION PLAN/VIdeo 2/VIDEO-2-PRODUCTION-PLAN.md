## SCRIPT STRUCTURE

### **HOOK (0:00 - 0:45)**
Reference Video 1 → Address viewer question → Set up the build → Promise a result

### **CONTEXT (0:45 - 2:00)**
The gap between "understanding AI" and "using AI" → Quick Video 1 recap → Transition to building

### **THE BUILD (2:00 - 9:30)**
- **Step 1: Define the Trigger** (3:00) — When does the workflow fire?
- **Step 2: Set the Context Document** (4:30) — The reusable "employee briefing"
- **Step 3: Chain the Tasks** (6:00) — Organize → Write → Refine (not one prompt)
- **Step 4: Test and Refine** (8:00) — Run it, fix what breaks, improve weekly
- **Result Reveal** (9:30) — Before vs. after comparison

### **CALLBACK/CTA (10:30 - 12:00)**
Recap four steps → Homework (pick one task, build tonight) → Subscribe → Tease Video 3 → Comment prompt

## PRODUCTION CHECKLIST

### Pre-Production
- [ ] Watch 2-3 Howfinity tutorials (note screen recording pacing)
- [ ] Watch 1 Igor Pogany walkthrough (note explanation layering)
- [ ] Build the actual workflow yourself first — know the steps cold
- [ ] Record voice memo explaining the workflow out loud
- [ ] Finalize script — time it out loud (should be 9-11 min spoken)
- [ ] Plan screen demos: what tool, what inputs, what outputs at each step
- [ ] Prepare "before vs. after" comparison screenshots

### Production
- [ ] Film talking head sections (hook, transitions, CTA)
- [ ] Record screen demos for each step (do each 2x — pick cleaner take)
- [ ] Capture "result reveal" moment (the payoff shot)
- [ ] Film the "before" scenario (vague chatting, frustration)

### Post-Production
- [ ] Edit: tighten screen demos (cut dead air, zoom into key areas)
- [ ] Add step number overlays (STEP 1, STEP 2, etc.)
- [ ] Add text overlays for key concepts
- [ ] Create thumbnail (test 2-3 options)
- [ ] Write description, tags, and timestamps
- [ ] Add end screen linking back to Video 1

### Launch Day
- [ ] Post Twitter/X thread
- [ ] Create TikTok/Shorts vertical clip (Aleric Heck style)
- [ ] Pin comment linking to Video 1
- [ ] Reply to ALL comments in first 24 hours
