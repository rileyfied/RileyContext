CFA Merritt Island Training Arcade App — standalone HTML app, arcade/quiz game collection for the CFA team. All games in one place. Inspired by every popular beautiful quiz/game app that has existed.

Game ideas so far:
- Priority Tap: tap tasks in correct priority order (e.g. clean in place, greet guest, stock, change paper towels, wash hands)
- Core Values Order: put the 7 CFA core values in correct order
- Lemonade Recipe: how do you make lemonade step by step
- Ring It In: how do you ring in [item] correctly
- Car Logo Game (multiple variants): identify car logos, make/model/color
- If/Then Scenarios: what do you do if X happens — covers edge cases and "what do you do if you don't know"
- Excellence Game: "You're doing great! But are you Excellent?" — teaches nuance of Striving for Excellence, one of CFA's core values

Design vibe: arcade-style, animated, competitive, beautiful. NOT generic. Every game should feel like a real game.

Knowledge base needs to be built via Q&A sessions with Riley — all answers are Merritt Island-location-specific, not generic CFA SOP. Plan is for Riley to do dictation/stream-of-thought Q&A sessions to capture the correct answers.

Also discussed: shift hype board popup (team names on screen at shift start, motivational line, station spotlight) — possibly integrated as a non-game screen in the same app. #cfa #training #carspotter #arcade
