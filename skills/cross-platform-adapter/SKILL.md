# Cross-Platform Content Adapter

Transform YouTube content into native content for Twitter/X and TikTok. Not cross-posting—true platform-native adaptation.

## Core Principle

Each platform has its own grammar, attention economy, and culture. Content that works on YouTube rarely works copy-pasted elsewhere. This skill creates genuinely native content inspired by YouTube source material.

## Workflow Overview

1. **Source Analysis** — Understand the YouTube content deeply
2. **Platform Research** — Check current trends on target platform
3. **Adaptation Strategy** — Decide approach for each platform
4. **Content Creation** — Produce platform-native outputs

## Phase 1: Source Analysis

Before adapting, extract the core elements:

**Content inventory:**
- Main thesis/takeaway (one sentence)
- Key moments (timestamps of highest-value segments)
- Quotable lines (things that stand alone)
- Visual highlights (screenshots, demonstrations worth capturing)
- Story/narrative hooks (personal anecdotes, surprising reveals)

**Adaptation potential assessment:**
- What's the most shareable single idea?
- What moment would make someone want to watch the full video?
- What's counterintuitive or surprising?
- What solves an immediate problem?

## Phase 2: Platform Research

Research current platform state before adapting. Use web_search:

### For Twitter/X
- `Twitter AI content viral 2026` — What formats are working
- `Twitter thread examples [topic]` — Structure patterns
- `Twitter engagement strategies` — Current algorithm preferences

**Platform context to gather:**
- Trending formats (threads vs. single tweets vs. quote tweets)
- Optimal posting times for this audience
- Hashtag relevance (often declining—verify)
- Current character/media limits

### For TikTok
- `TikTok AI content viral 2026` — What's performing
- `TikTok trends this week` — Current sounds/formats
- `TikTok hook examples` — Opening patterns that work

**Platform context to gather:**
- Trending sounds relevant to topic
- Video length sweet spots
- Current editing styles
- Hook patterns that stop the scroll

See `references/platform-specs.md` for technical specifications.

## Phase 3: Adaptation Strategy

Choose approach based on source content and platform:

### Twitter/X Strategies

**Thread adaptation** — Break video into sequential insights
- Best for: Tutorials, explainers, multi-step processes
- Structure: Hook tweet → Key points → CTA to video

**Single tweet extraction** — Pull one powerful moment
- Best for: Hot takes, surprising stats, quotable insights
- Structure: Bold claim + context + link

**Quote tweet bait** — Create something people want to respond to
- Best for: Opinions, predictions, controversial takes
- Structure: Provocative framing that invites discussion

**Carousel/image post** — Visual summary of key points
- Best for: Comparisons, step-by-step, before/after
- Structure: Eye-catching first image → Value slides → CTA

See `references/twitter-templates.md` for detailed templates.

### TikTok Strategies

**Clip extraction** — Pull highest-energy 30-60 seconds
- Best for: Demonstrations, reactions, reveals
- Edit: Vertical crop, captions, trending sound

**Re-record hook** — New intro optimized for TikTok attention
- Best for: Tutorials, tips, explanations
- Structure: Pattern interrupt → Compressed value → Full video tease

**Duet/stitch bait** — Create content others want to respond to
- Best for: Opinions, challenges, questions
- Structure: Open loop that invites completion

**Series chunking** — Break into multi-part TikTok series
- Best for: Long-form tutorials, deep dives
- Structure: Part 1/2/3 with cliffhangers

See `references/tiktok-templates.md` for detailed templates.

## Phase 4: Content Creation

Produce the actual adapted content:

### Twitter/X Deliverables

For each adaptation, provide:
- Complete tweet/thread text (ready to copy-paste)
- Image suggestions or actual images if applicable
- Optimal posting time recommendation
- Engagement strategy (what to reply to comments with)

### TikTok Deliverables

For each adaptation, provide:
- Script with exact timing
- Hook (first 1-3 seconds)
- Caption text
- Sound recommendation (if applicable)
- Hashtag strategy
- Clip timestamps from source video (if extracting)

## Quality Checks

Before delivering adapted content, verify:

**Platform nativeness:**
- Would this feel out of place on the platform? If yes, rework.
- Does it follow current platform conventions?
- Is the tone right for the platform culture?

**Value independence:**
- Does this provide value even if they never watch the YouTube video?
- Is the CTA earned, not forced?

**Brand consistency:**
- Does this still sound like the creator?
- Does it maintain the accessible, non-tech-bro voice?
