# Twitter/X Templates

Ready-to-use structures for adapting YouTube content to Twitter.

---

## Thread Templates

### Tutorial Thread
```
🧵 How to [achieve outcome] in [timeframe/steps]

(A thread)

↓
```

**Tweet 2-N (one per step):**
```
[Step number]. [Action]

[1-2 sentences explaining why/how]

[Optional: screenshot or example]
```

**Final tweet:**
```
That's it. [Outcome] in [X] steps.

Full walkthrough with demos: [link]

If this helped, RT the first tweet so others can find it.
```

---

### Insight Thread
```
I just [discovered/realized/tested] something about [topic].

Here's what I found:

🧵
```

**Tweet 2-N (one per insight):**
```
[Insight number/emoji]

[The insight in 1-2 sentences]

[Why it matters or example]
```

**Final tweet:**
```
TL;DR:
• [Key point 1]
• [Key point 2]
• [Key point 3]

Deep dive on this: [link]
```

---

### Contrarian Thread
```
Popular opinion: [common belief]

Unpopular opinion: [your take]

Here's why 👇
```

**Supporting tweets:**
```
[Evidence/reasoning point 1]

[Evidence/reasoning point 2]

[Evidence/reasoning point 3]
```

**Conclusion:**
```
This doesn't mean [strawman].

It means [nuanced take].

I made a whole video breaking this down: [link]
```

## Single Tweet Templates

### Bold Claim
```
[Surprising statement about topic].

[One sentence of context/proof].

[Link or CTA]
```

**Example:**
```
Most people use AI wrong.

They ask it questions. They should give it systems.

Here's what I mean: [link]
```

## Engagement Strategies

### Reply Prompts
Include these to drive comments:
- "What's your experience with this?"
- "What would you add?"
- "Hot take or common sense?"
- "Agree or disagree?"

### First Reply Strategy
Immediately reply to your own tweet with:
- Additional context that didn't fit
- A relevant resource
- A question to seed discussion

### Quote Tweet Bait
Frame content so others want to add their take:
- Make a claim people will want to validate or dispute
- Ask an open-ended question
- Share something incomplete that invites completion

## Character-Saving Techniques

- Use "→" instead of "leads to"
- Use "&" instead of "and"
- Use numerals (5) not words (five)
- Use line breaks for visual clarity (don't count against limit)
- Use em-dash (—) to connect thoughts without period

## Hashtag Guidelines

**Current state (2026):** Hashtags are low-signal. Use sparingly if at all.

**When to use:**
- Joining an active conversation (#topic when trending)
- Niche communities that actively search hashtags

**When to skip:**
- General content (algorithm handles discoverability)
- Multiple hashtags (looks spammy)

**If using:** 1-2 max, at end of tweet, relevant and specific
