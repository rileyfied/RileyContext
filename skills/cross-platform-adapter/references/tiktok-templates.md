# TikTok Templates

Structures and scripts for adapting YouTube content to TikTok.

---

## Hook Templates

The first 1-3 seconds determine everything. Pattern interrupt required.

### Direct Challenge
```
"Stop [common behavior]."
"[Number] things you're doing wrong with [topic]."
"Nobody talks about this but..."
```

### Outcome Lead
```
"This is how I [achieved result]."
"I [did thing] and here's what happened."
"[Time] ago I couldn't [thing]. Now I [result]."
```

### Curiosity Gap
```
"The reason [surprising thing] is..."
"Wait until you see what happens when..."
"I wasn't supposed to share this but..."
```

### Controversy/Hot Take
```
"Unpopular opinion: [take]"
"I'm gonna get hate for this but..."
"Everyone says [thing] but actually..."
```

### Question Hook
```
"Why does no one talk about [thing]?"
"Have you ever noticed [observation]?"
"What if I told you [surprising claim]?"
```

```
[HOOK - 2 sec]

"When you [situation]..."

[Show/demonstrate]
```

## Sound Strategy

### When to Use Trending Sounds
- Content fits the vibe naturally
- Sound is still rising (not oversaturated)
- Your audience expects/enjoys trends

### When to Use Original Audio
- Tutorial/educational content
- Personal story that needs your voice
- Building recognizable brand audio

### Sound Discovery
- Check TikTok's "trending sounds" weekly
- Note what sounds appear in your niche
- Save sounds when you encounter good fits

## Clip Extraction Guide

When pulling clips from YouTube videos:

### What Makes a Good Clip
- Self-contained value (doesn't need context)
- Strong visual component
- Energy peak moment
- Surprising or counterintuitive statement
- Clear demonstration with visible result

### What to Avoid
- Clips that require YouTube context
- Low-energy explanations
- References to "earlier in the video"
- Long setup before payoff

### Technical Extraction
1. Identify timestamp range
2. Export at 9:16 (crop/reframe for vertical)
3. Add captions (essential for TikTok)
4. Add hook text overlay if needed
5. Consider adding trending sound underneath (low volume)

## Engagement Tactics

### Comment Section Strategy
- Reply to comments within first hour
- Pin a question to encourage more comments
- Reply with follow-up questions to extend threads
- Use comments to tease next video

### Duet/Stitch Bait
Create content that invites responses:
- "I can't be the only one who..."
- "POV: You just discovered..."
- Open-ended questions
- Controversial takes (within brand)

### Save/Share Optimization
- Promise value worth returning to ("Save this for when...")
- Create completeness (full tutorial in one video)
- Make it useful enough to share with friends
