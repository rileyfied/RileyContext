# Platform Specifications

Technical specs and constraints for each platform. Updated periodically—verify current limits if creating time-sensitive content.

---

## Twitter/X

### Text Limits
- Single tweet: 280 characters (standard), 25,000 characters (Premium)
- Thread: No hard limit, but 5-12 tweets optimal for engagement
- Alt text: 1,000 characters

### Media Specs
- Images: Up to 4 per tweet, 5MB max each, 4096x4096 max resolution
- GIFs: 15MB max, 1280x1080 recommended
- Video: 2:20 max (standard), 60 min (Premium), 512MB max
- Aspect ratios: 16:9 (landscape), 1:1 (square), 9:16 (vertical all supported)

### Link Behavior
- Links consume ~23 characters regardless of actual length
- Link previews show for single links (suppressed with multiple links)
- Links in replies often perform better than in original tweet

### Algorithm Considerations (2026)
- Replies and quote tweets weighted heavily
- First hour engagement critical
- Video watch time now factored
- Threads distributed tweet-by-tweet in feeds

## TikTok

### Video Specs
- Length: 15 sec to 10 min (60 sec optimal for most content)
- Aspect ratio: 9:16 (vertical) required for best performance
- Resolution: 1080x1920 recommended
- File size: 287.6 MB max (iOS), 72 MB max (Android)
- Format: MP4 or MOV

### Text/Caption
- Caption: 4,000 characters max
- On-screen text: Keep in safe zone (away from edges)
- Auto-captions available but often need correction

### Sound
- Original audio or trending sounds
- Trending sounds boost discoverability
- Original audio can become trending if video performs

### Hook Requirements
- First 1-3 seconds determine swipe-away
- Pattern interrupt essential
- Text hook often appears before spoken hook

### Algorithm Considerations (2026)
- Watch time percentage most important signal
- Completion rate critical for short videos
- Shares weighted heavily
- Comments boost reach significantly
- "Not interested" signals hurt more than low engagement

## Cross-Platform Posting Considerations

### Timing
- Twitter: 8-10am, 12-1pm, 5-6pm (weekdays) in target timezone
- TikTok: 7-9am, 12-3pm, 7-11pm (varies by audience)
- Post TikTok first if repurposing (algorithm rewards native content)

### Watermark Issues
- TikTok watermarks hurt performance on other platforms
- Download without watermark before cross-posting
- Instagram Reels explicitly deprioritizes TikTok watermarks

### Caption/Hashtag Differences
- Twitter: Hashtags increasingly optional, 1-2 max if used
- TikTok: 3-5 relevant hashtags still valuable, mix popular + niche

## Content Format Matrix

| YouTube Content Type | Best Twitter Format | Best TikTok Format |
|---------------------|--------------------|--------------------|
| Tutorial (10+ min) | Thread or carousel | Series or re-recorded hook |
| Hot take/opinion | Single tweet or quote-tweet bait | Direct-to-camera clip |
| Demo/showcase | GIF + context tweet | Clip extraction with captions |
| Interview/conversation | Quote extraction thread | Best moment clip |
| News/reaction | Single tweet with take | Stitch-style reaction |
| Deep dive | Thread with summary | Part 1/2/3 series |
